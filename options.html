Custom Replacements

Enter custom replacement rules below. There should be one per line, in the form FROM -> TO, e.g. \therefore -> ∴. If either the FROM or TO sections of the rule contain the text ->, it should be written as -\>, and \ should be written as \\. The space around the -> is required.

Custom Commands

Custom commands are single-argument functions accept and return text. A custom function \f is invoked like \f{input text}. To create your own custom commands, write JavaScript code below, calling defCommand(commandName, command) for ech command. For instance, `defCommand("\\uppercase", text => text.toUpperCase())` creates a command that turns text uppercase; it would then be used like \uppercase{some text} which would produce SOME TEXT.

Settings

Timeout

Timeout is the tolerance to slow typing. It is the number of milliseconds of no typing tolerated, after which the current command will end. So if you type \alp, wait a bit, and then type ha, the command \alpha will only be invoked if the time you waited was ≤ timeout.

Lookbehind

Lookbehind is the tolerance to fast typing. It is the number of characters that will be checked behind the cursor for macros. Sometimes, a keypress will be missed, so that you may type \alpha but it is not recognized. Lookbehind means that if you continue typing, say, typing a space to result in \alpha , when you type the space, the extension will go back and re-check \alpha, recognize it, and invoke it.

Some of the extension's functionality requires lookbehind to be 1 or above, so don't set it below 1.
