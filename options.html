Custom Replacements

Enter custom replacement rules below. There should be one per line, in the form FROM -> TO, e.g. \therefore -> ∴. If either the FROM or TO sections of the rule contain the text ->, it should be written as -\>, and \ should be written as \\. The space around the -> is required.

Custom Commands

Custom commands are single-argument functions accept and return text. A custom function \f is invoked like \f{input text}. To create your own custom commands, write JavaScript code below, calling defCommand(commandName, command) for ech command. For instance, `defCommand("\\uppercase", text => text.toUpperCase())` creates a command that turns text uppercase; it would then be used like \uppercase{some text} which would produce SOME TEXT.
